# Vite + React + Shadcn + Typescript

This is a template for a Vite.js, React, and Shadcn UI project with Typescript.

## Getting Started

### Prerequisites

- Node.js
- npm

### Installing

1. Clone the repository
2. Install the dependencies

```
npm install
```

### Running the project

```
npm run dev
```

This will start the development server.

### Building the project

```
npm run build
```

This will build the project for production.

## Features

- Vite.js
- React
- Shadcn UI
- Typescript
- Supabase
